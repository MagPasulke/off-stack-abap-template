# abapGit Object Creation Skills

Skills for creating abapGit-serialized SAP objects directly as repository files, without requiring an SAP system.

Each skill provides:
- abapGit serialization rules (UTF-8/BOM, LF, 2-space indent)
- File structure and naming conventions
- Complete generic samples (`ZFOO_*` naming) with full XML/ABAP content
- Key XML elements reference table
- Creation workflow and done-check

> **abapGit docs**: [Supported Object Types](https://docs.abapgit.org/user-guide/reference/supported.html) | [File Formats](https://docs.abapgit.org/development-guide/serializers/file-formats.html) | [Folders & Filenames](https://docs.abapgit.org/user-guide/reference/folders-filenames.html)
